# Cosmetic_Intelligent_Assisstant# AI Cosmetic Intelligence Agent

## Overview

An AI expert for makeup and skincare companies. Combines real-time web data with regulatory knowledge for product innovation, FDA compliance, marketing strategies, and consumer trends.

## Features

- Discover viral TikTok/Instagram beauty trends
- Review ingredient safety and allergy risks
- Check FDA, EU, ASEAN compliance
- Generate product names and descriptions
- Recommend products by skin type
- WebSearchTool for real-time data

## Tech Stack

- Python 3.12+, OpenAI Agents SDK, Chainlit, WebSearchTool, python-dotenv

## Setup

1. cd Cosmetic_Intelligent_Assisstant
2. pip install -r requirements.txt
3. Copy .env.example to .env and set OPENAI_API_KEY

## Run Commands

chainlit run app.py

## Example Use Cases

1. Suggest a sustainable skincare product based on Instagram trends and FDA compliance
2. Review cosmetic ingredient safety for sensitive skin
3. Recommend products for oily skin with eco packaging
